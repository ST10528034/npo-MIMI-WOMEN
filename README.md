#Website for npo-MIMI-WOMEN

this website is created to supprt and promote women empowerment by providing information, resources, and services tailored for women. the website aims to create awareness, encouragement engagement, and provide an accessible platform for users to interact with the organisation. it is developed as part of a web development part 1, focusing on HTML structure, usability, and user experience.

the reason i created this website is to inform users about Mimi Women and its mission. to provide access to services and resources for women. to promote engagement through a clean and user-friendly interface. to demonstrate web development skills using HTML

I included the home page, about us page, contact page, products and services page, gallery page, sponsors page, cart and a search page. i used HTML for structuring content and Github for version control and project management.

the website is designed to be easy to navigate, present information quickly and maintain consistent design across all pages.
